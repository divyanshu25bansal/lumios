# 🌟 Lumios

> **Your AI-powered wellness companion designed to help you build healthier habits, stay hydrated, and improve your overall well-being through intelligent insights and voice interactions.**

---

## 🚀 Project Overview

Lumios is a modern AI-driven health platform that combines wellness tracking with conversational AI.

It helps users:

* 💧 Track daily hydration goals
* 🎯 Build and maintain healthy habits
* 📊 Monitor progress through analytics
* 🤖 Receive personalized AI wellness coaching
* 🎙️ Interact naturally through voice
* 📈 Gain actionable insights from health data

> **⚠️ This project is currently under active development.**

## 📦 Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Open Application

```txt
http://localhost:3000
```
